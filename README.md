# Anti-Fruad 反诈提醒页面
需要 PHP 环境支持，最低需求版本我没有测试过，我使用的 PHP 版本为 8.3。  
将 `counter.php` `counter.txt` `index.php` 直接放到对应网站目录即可。  
当访问 `index.php` 的时候，会调用访问 `counter.php` ，将 `counter.txt` 内记录的数值增加 1。

英语 English  
# Anti-Fraud Reminder Page  
Requires PHP environment support. Minimum required version untested; tested with PHP 8.3.  
Place `counter.php`, `counter.txt`, and `index.php` directly in the corresponding website directory.  

When `index.php` is accessed, it accesses `counter.php`, incrementing the value recorded in `counter.txt` by 1.
